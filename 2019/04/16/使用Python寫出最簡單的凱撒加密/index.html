vick

Python, 密碼學

什麼是凱撒加密

它我知道最簡單的加密法，其實原理很簡單，將密碼按照Ascii碼偏移一個指定的量之後，就會被轉換成加密過後的密碼了

如何加密

以英文來說，我將它們都偏移一位
原密碼：ABCDE
加密後：BCDEF

這樣就是最簡單的凱撒加密了
接下來換成其他密碼試試
原密碼：mypassword
加密後：nzqbttxpse
這樣效果看起來就比較明顯了

當然也不一定只能偏移一位，但是有個先決條件，加密偏移量多少解密就要多少

如何解密

至於解密怎麼做，非常簡單就只要把加密的動做倒過來就好
怎麼倒過來呢？
原本是A -> B, B -> C …
現在就反過來變成 B-> A, C -> B
