vick
2020-02-15
VS Code

前言

今天這篇算是寫給我自己看的筆記，就簡單紀錄下來，方便之後忘記回來看或是幫到其他有需要的人，此篇的設定主要是針對windows系統去做設定，前提要有裝好git，這邊就先不談怎麼安裝，之後有機會再補充。

本文

打開設定檔

首先按下 ctrl + ~ 打開terminal，在沒改過設定應該會像圖片中一樣套用powershell

接下來我們要打開json設定檔，以下有兩種方式可以開啟，擇一即可

方式一

按下 ctrl + shift + p 打開輸入命令的面版，輸入 Open Settings(JSON) 即可直接打開json設定檔

方式二

左上角的選單內選擇 File > Preferences > Settings 會進入UI的設定，再往下找到隨便一個 Edit in settings.json 的連結就會一樣直接打開json設定檔，不用像圖片中一定要到 Features > Terminal 裡面去 （只是懶得重新截圖）

完成
