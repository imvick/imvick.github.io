如何用C#做到開頭大寫

vick
2020-06-30
C#

前言

這次會分享兩種方法來達成開頭大寫，兩種方法的核心概念都很簡單，就是先將第一個字取出來轉成大寫，再將之後的字組合起來就完成了。

方法一、利用 Char陣列

大致流程如下：

  1. 將string轉成char陣列
  2. 將char陣列的第一筆資料（char[0]）用ToUpper()轉大寫
  3. 最後再利用string的建構子將原先的char陣列傳入
  4. 取得結果，完成

方法二、利用 Substring

大致流程如下：

  1. 使用Substring(0, 1)抓到第一個字，再利用ToUpper()轉大寫
  2. 再使用Substring(1)代表抓第一個字之後的所有字
  3. 將前兩個步驟結果組合在一起就完成了

兩種方法的效能差異

使用上面兩個範例放在迴圈去跑了一千萬次次計算出來的結果，
我也不知道這樣計算的結果是否精準，不過我就放上來讓大家作為一個參考依據

  • 方法一 花了 562ms
  • 方法二 花了 1446ms

兩者的花費時間差距約為2.5倍（這邊測試僅供參考，方便比較出兩種方法的Performance）。
